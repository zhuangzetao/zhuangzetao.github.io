    hugo搭建 #

    创建网站

    hugo new site fhugo
cd fhugo
git init
#拉取主题
git submodule add https://github.com/tohn/linkshrubbery.git themes/linkshrubbery

    添加内容

    hugo new post/first.md

    本地启动个人博客

    hugo server -t hugo-book-master --buildDrafts
